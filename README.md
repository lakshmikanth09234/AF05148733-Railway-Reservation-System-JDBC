# Railway Reservation System (Java JDBC)

This is a console-based Railway Reservation System developed using Java and JDBC with MySQL database integration.

## 🚀 Features
- Book train tickets
- Cancel reservations
- View available trains
- Passenger details management

## 🛠 Technologies Used
- Java
- JDBC
- MySQL

## 🧠 Concepts Used
- Database connectivity
- CRUD operations
- Menu-driven console application

## ▶️ How to Run
1. Set up MySQL database
2. Update DB credentials in code
3. Run Main.java

## 📌 Note
This is a beginner-level project to understand database operations using Java.
